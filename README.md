
# Movie App

Movie App is a module built with React, Redux, and Vite. It uses the following dependencies: 

- `@reduxjs/toolkit` 
- `animate.css` 
- `axios` 
- `moment` 
- `normalize.css` 
- `react` 
- `react-dom` 
- `react-icons` 
- `react-loading` 
- `react-redux` 
- `react-router-dom` 
- `react-slick` 
- `slick carousel`  
and  
 - `styled components`.  

 The devDependencies include:  

 - `@types/react`,  

 - `@types/react dom`,  

 - `@vitejs/plugin react`, and   

 - `vite`.
